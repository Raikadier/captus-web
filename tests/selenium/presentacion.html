¿Qué es Selenium?

  • 1 Herramienta open source para automatizar navegadores web
  • 2 Creado en 2004 por Jason Huggins en ThoughtWorks
  • 3 Permite controlar Chrome, Firefox, Edge y Safari desde código
  • 4 Soporta Python, Java, JavaScript, C#, Ruby
  • 5 Estándar de la industria para pruebas end-to-end

"Si un usuario puede hacerlo en el browser, Selenium también puede hacerlo."

Acciones principales

from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

driver = webdriver.Chrome()            # abre el navegador
driver.get("http://localhost:5173")     # navega a una URL

campo = driver.find_element(By.ID, "email")  # encuentra elemento
campo.send_keys("usuario@test.com")           # escribe texto
campo.click()                                   # hace click

# Esperar a que aparezca un elemento (máx 10 seg)
WebDriverWait(driver, 10).until(
    EC.visibility_of_element_located((By.CSS_SELECTOR, ".text-red-600"))
)
driver.quit()                               # cierra el navegador

Cómo encontrar elementos

# Por ID  →  la forma más rápida y recomendada
driver.find_element(By.ID, "email")

# Por selector CSS  →  muy potente y expresivo
driver.find_element(By.CSS_SELECTOR, "button[type='submit']")

# Por XPath  →  para estructuras complejas
driver.find_element(By.XPATH, "//button[contains(text(), '¿No tienes cuenta?')]")

# Por NAME, CLASS_NAME, TAG_NAME también existen
driver.find_element(By.NAME, "password")

Regla de oro: prefiere ID → CSS → XPath

Test 5 — Credenciales inválidas

def test_05_error_con_credenciales_invalidas(self):
    # 1. Escribir email y contraseña incorrectos
    self.driver.find_element(By.ID, "email").send_keys("nadie@noexiste.xyz")
    self.driver.find_element(By.ID, "password").send_keys("claveInvalida999")

    # 2. Hacer click en el botón de enviar
    self.driver.find_element(By.CSS_SELECTOR, "button[type='submit']").click()

    # 3. Esperar hasta que aparezca el mensaje de error
    error = self.wait.until(
        EC.visibility_of_element_located((By.CSS_SELECTOR, ".text-red-600"))
    )

    # 4. Afirmar que el error está visible
    self.assertTrue(error.is_displayed())

¿Cuándo usar Selenium?

✅

Úsalo para…

Flujos completos de usuario: login, registrar, navegar.
Verificar que nada se rompe tras un cambio.
Formularios y validaciones visuales.

❌

No lo uses para…

Lógica interna de funciones (usa unit tests).
Pruebas de APIs (usa Postman o pytest + requests).
Código que cambia muy rápido.

Selenium es la última línea de defensa: prueba lo que el usuario realmente ve.
